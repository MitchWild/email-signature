How to Add Your Signature in Outlook

Outlook Desktop (Windows)

  1. Fill in your information above and click "Copy Signature"
  2. In Outlook, go to File → Options → Mail → Signatures
  3. Click "New" and name it (e.g., "WI Aviation")
  4. Click inside the signature editor box and press Ctrl+V to paste
  5. Set it as your default for new messages and replies, then click OK

Outlook on the Web (outlook.office.com)

  1. Fill in your information above and click "Copy Signature"
  2. Click the gear icon (top right) → View all Outlook settings
  3. Go to Mail → Compose and reply
  4. Under "Email signature," click in the editor and press Ctrl+V to paste
  5. Check both boxes to use it for new messages and replies, then Save

Outlook for Mac

  1. Fill in your information above and click "Copy Signature"
  2. In Outlook, go to Outlook → Settings → Signatures
  3. Click "+" to create a new signature
  4. Click in the editor and press Cmd+V to paste, then close

Adding a Custom Badge Image

  1. Email signatures need images hosted online — you can't attach a file from your computer
  2. If your image is already on a website, right-click it, select "Copy image address", and paste the URL into the Custom Badge field above
  3. If you have an image file that isn't online yet, send it to Mitchell Wild at mitchw@wisav.com and he'll upload it and give you a URL to paste in
Tip: If images don't appear right away, make sure you're connected to the internet. Outlook loads signature images from the hosted URLs each time. If anything looks off, re-copy the signature from this page and paste again.
